# 币讯AI - 区块链投资信息平台

币圈资讯一站式聚合平台，提供AI新闻摘要、行情监控、链上数据分析等服务。

## 产品定位

- **目标用户：** 国内币圈投资者
- **核心价值：** 解决信息差问题，聚合币圈资讯
- **商业模式：** 订阅制（19/月 ~ 49/月）

## 功能规划

### MVP功能
- [ ] AI新闻摘要
- [ ] 行情概览（BTC/ETH/主流币）
- [ ] 重要公告监控
- [ ] AI问答

### 第二阶段
- [ ] 链上数据分析
- [ ] 持仓提醒
- [ ] 项目解读
- [ ] 社群功能

## 技术栈

- **前端：** 微信小程序（Uniapp/Taro）
- **后端：** Node.js/Python
- **AI：** Claude API / GPT-4 API
- **数据源：** Binance API, X API, Brave API
- **服务器：** 海外轻量云服务器
